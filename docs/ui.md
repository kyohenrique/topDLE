# UI

## Visual Direction

- NVIDIA-inspired dark-first look
- Black/near-black surfaces
- Green accent (`#76b900`) as interaction highlight
- Sharp corners (`2px`) and compact spacing

## Layout

- Header with search/sort/language/theme controls
- Card grid:
  - mobile: 1 column
  - desktop: 2 columns
- Footer with GitHub icon and author credit

## Card Behavior

- Entire card is clickable
- Local logo image
- Description appears on hover/focus overlay

## Responsiveness

- Inputs and controls collapse cleanly on small screens
- Grid adapts without layout shift
